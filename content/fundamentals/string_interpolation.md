String interpolation is a process substituting values of variables into placeholders in a string.

### Python

```python
# f-Strings
print( f"the total of {a} + {b} is: {a + b}" )
print( f"{name} is {age}" )

# str.format()
print( "{} is {}".format(name, age) )

# string formatting operators
print( "%s is %d" % (name, age) )
```

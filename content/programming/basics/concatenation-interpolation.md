| Language                | Concatenation Operator |
| ----------------------- | ---------------------- |
| JavaScript / TypeScript | `+`                    |
| PHP                     | `.`                    |

### PHP

Concatenation operator is `.`  
Escape character is `\`

```php
<?php
$name = "Amna";
$age = 32;

// Concatenation
echo $name . ' is ' . $age . ' year(s) old.';
?>
```

- Disk: `df -h`
- RAM: `free -m`
- CPU: `ps -ef  / top`
- Load Averages: `uptime`

### Memory Usage

- `top` Show top processes
- `free` Display amount of free and used memory in the system.
- `cat /proc/meminfo` Show memory size and usage.
- `vmstat` Monitor virtual memory

title: Core Modules
date: 2016-08-19
lastmod: 2017-03-13
---

core modules are the ones that are there, you don't have to install them, but you do need to specifically include them. For example, load the file system module:

```javascript
var fs = require("fs")
```

OR import the ES6 way

```javascript
import fs from 'fs'
```
